# moptracker

En enkel, fristående webbapplikation för rapportering av **ED digital utbildningsstruktur**.

## Innehåll
Projektet består i nuläget av en HTML-fil:

- `RapporteringEDDigital.html` – innehåller både struktur, styling och klientlogik.

## Kom igång
Eftersom applikationen är statisk kan du öppna filen direkt i webbläsaren.

### Alternativ 1: Öppna filen direkt
1. Dubbelklicka på `RapporteringEDDigital.html`.
2. Applikationen startar i din standardwebbläsare.

### Alternativ 2: Kör via lokal server
Om du vill undvika eventuella begränsningar i webbläsaren kan du starta en enkel lokal server:

```bash
python3 -m http.server 8000
```

Öppna sedan:

```text
http://localhost:8000/RapporteringEDDigital.html
```

## Syfte
README-filen gör projektet enklare att förstå och komma igång med för nya användare.
